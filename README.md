# Odisha Startup Growth Analysis (FY 2021-22)

Done as part of my internship at Startup Odisha.
The dataset covers 826 startups across Odisha for fiscal years 2021 and 2022.

## What I did
- Computed profit scores and revenue growth rates for all 826 startups
- Identified top 10 startups by total revenue and profit
- Found that women-led startups grew 106% — double the overall 53% average
- Used KMeans to consolidate 44 industry types into 13 economic clusters
- Built a Linear Regression model to forecast revenue through 2025
- Exported top 10 startup revenue forecasts as CSV

## Key numbers
- 826 startups analyzed
- 53% overall revenue growth
- 106% women-led startup growth
- 44 industries → 13 clusters
- Revenue forecasted through FY2025

## How to run
Place the CSV file in the same folder and run the notebook top to bottom.

## Stack
Python · Pandas · scikit-learn · Matplotlib · Seaborn · KMeans
